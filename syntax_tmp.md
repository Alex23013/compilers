# Languaje syntax

### Variables
```
type name
type name = value
char name = "a"
```

### Array:
```
type[] name 
type[] name = [value, value, value, ...]
```

### String
```
char[] name
char[] name = "value"
char[] name = ["v","a","l", ...]
```

### Function
```
func name(type name, type name, ...) : return_type {
  ...
  return value
}
```

### Control
```
if (something) {
  ...
}
```

```
else {
  ...
}
```

```
while (something) {
  ...
}
```

### Additional
```
type name1, name2, ...
type name1 = value1, name2 = value2, ...
```
