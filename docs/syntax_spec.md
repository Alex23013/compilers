# Languaje syntax

### Variables
```
type name
type name = value
string name = "a"
```

### Array:
```
type[] name 
type[] name = [value, value, value, ...]
```

### Function
```
func name(type name, type name, ...) : return_type {
  ...
  return value
}
```

### Control
```
if (something) {
  ...
}
```

```
elif (something) {
  ...
}
```

```
else {
  ...
}
```

```
while (something) {
  ...
}
```
